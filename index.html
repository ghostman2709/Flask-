HyperFoco - Comandos MikroTik Interativos

Clique no botão copiar para copiar o comando desejado.

Comandos Básicos

/ip address print
Lista todos os IPs configurados nas interfaces.
/ip cloud print
Mostra o IP público e o nome DNS do MikroTik Cloud DDNS.
/ip dhcp-server lease print
Lista os dispositivos que receberam IP via DHCP.
/ip firewall nat print
Exibe as regras de NAT configuradas no roteador.

Gerenciar SSH Remoto

/ip firewall filter add chain=input action=accept protocol=tcp dst-port=22 place-before=10 comment="Liberar SSH remoto temporariamente"
Libera o acesso SSH remoto temporariamente (coloque antes da regra de bloqueio).
/ip firewall filter remove [find comment="Liberar SSH remoto temporariamente"]
Remove a regra que libera o SSH remoto temporário.
/ip firewall filter disable [find comment="Liberar SSH remoto temporariamente"]
Desabilita a regra de SSH remoto temporário sem removê-la.

Gerenciar Usuários

/user add name=usuario_limitado group=read password=SENHA123
Adiciona um usuário limitado (somente leitura) com senha.
/user remove usuario_limitado
Remove o usuário limitado criado.

Gerenciar Tempo (Exemplo de Script para Desabilitar SSH após 30 minutos)

:
/system scheduler add name=desliga_ssh interval=30m on-event="/ip firewall filter disable [find comment=\"Liberar SSH remoto temporariamente\"]" start-time=startup
Cria uma tarefa agendada que desabilita a regra SSH temporária após 30 minutos.
