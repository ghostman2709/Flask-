HyperFoco - Comandos Básicos MikroTik

Guia rápido e organizado para você dominar o MikroTik via terminal.

1. Visualizar informações gerais

/ip address print             → Ver IPs configurados
/ip route print               → Ver rotas configuradas
/ip dns print                 → Ver configuração DNS
/ip dhcp-client print         → Ver cliente DHCP
/system identity print        → Ver nome do roteador

2. Internet e IP Público

/ip cloud print               → Ver IP público e DNS name (MyNetName)
/tool netwatch                → Monitorar conectividade

3. Leases DHCP

/ip dhcp-server lease print   → Ver dispositivos conectados (DHCP leases)

4. Firewall e NAT

/ip firewall filter print     → Ver regras de firewall
/ip firewall nat print        → Ver regras NAT

5. Acesso remoto e serviços

/ip service print             → Ver portas e serviços ativos (ex: SSH, Winbox)
/ip service disable telnet    → Desativar Telnet
/ip service enable ssh        → Ativar SSH

6. Ver interfaces e status

/interface print              → Ver todas interfaces
/interface wireless print     → Ver status do Wi-Fi

7. Liberar SSH remoto temporariamente

/ip firewall filter add chain=input action=accept protocol=tcp dst-port=22 place-before=10 comment="Liberar SSH remoto temporariamente"
